Strengthen Airbus' image on the web
CLAF deals with design, ergonomics, navigation and interactivity of the screen interfaces or of the web pages. It describes in a generic way how all the different types of pages for the portals, contents and services are introduced to the users. CLAF has been thought up to allow, with respect to the rules, the expression of each channel identity and to allow, according to the audience and purpose, the look and feel of eSites.

It is the reference frame, which is common to all the applications developed under Plumtree and to all these of the partners, to the existing or incoming web sites or mini web sites, to html pages. It describes in a generic way how all the different types of pages for the portals, contents and services are introduced to the users. 		 
Subtitle (subtitle H1)
CLAF's main goal is to harmonize the display of the different existing portals and eSites . It is a visual and functional signature, which expresses Airbus Industry's personality through screen interfaces. It also allows to guarantee a using environment, which is durably normalized, recognizable, consistent in image terms and successful in using terms and this for all the users.
